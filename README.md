Un script automatizado de alto rendimiento para GNU/Linux diseñado para gestionar y convertir ISOs de Xbox 360 de forma nativa.

## 🛠️ Instalación y Uso

1. Clona este repositorio y entra al directorio:
   git clone https://github.com/MAGU-Terix/conversor-Xbox-magu.git
   cd conversor-Xbox-magu

2. Verificación de Integridad (Recomendado):
   Copia y pega este comando en tu terminal para confirmar que el script es auténtico y no ha sido manipulado:
   echo "237ca4162e2cac8b522ce0d557f1ff54c49bc6d7fc39e25d49d2db8c85761784 conversor-Xbox-magu.sh" | sha256sum --check

3. Permisos y ejecución:
   Una vez verificado, asigna permisos de ejecución y lanza el script:
   chmod +x conversor-Xbox-magu.sh
   ./conversor-Xbox-magu.sh
