# 🎮 Conversor Xbox 360 - Edición Linux PRO
Desarrollado por **MAGU-Terix** 🐧🔥

Un script automatizado de alto rendimiento para GNU/Linux diseñado para gestionar y convertir ISOs de Xbox 360 de forma nativa.

## 🛠️ Instalación y Uso
1. Clona este repositorio:
   git clone https://github.com/MAGU-Terix/conversor-Xbox-magu.git
   cd conversor-Xbox-magu

2. Permisos y ejecución:
   chmod +x conversor-Xbox-magu.sh
   ./conversor-Xbox-magu.sh

## 🔒 Verificación de Integridad (SHA-256)
Para garantizar la seguridad y autenticidad del script, puedes verificar su integridad ejecutando el siguiente comando en tu terminal:

sha256sum conversor-Xbox-magu.sh
